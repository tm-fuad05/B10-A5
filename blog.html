Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a tree-like structure that represents the elements of a web page, allowing scripts to manipulate its content and structure dynamically.

How do you select an element from the DOM?

Use methods like getElementById(id), getElementsByClassName(className),or querySelector(selector). For example, document.getElementById('myElement').

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is adding a single event listener to a parent element to handle events for its children. It's useful for improving performance and managing dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

Use setAttribute(name, value) to change attributes, and style.property = value to change styles. For example, element.setAttribute('src', 'image.png') and element.style.color = 'red'.
